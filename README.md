# web-mastery — a Claude Code skill for expert web design

A research-grounded skill that teaches Claude to build websites that look expert-made and actually convert. Distilled from Baymard Institute, the Stanford Web Credibility Project, the 50ms first-impression studies, Core Web Vitals business cases (Google/Deloitte, Rakuten, eBay), Laws of UX, Finsweet Client-First, and hard-won lessons from real Webflow API builds.

## What it covers

| File | Contents |
|---|---|
| `SKILL.md` | The 11 operating laws, standing copy/design rules, and the build workflow |
| `references/success-science.md` | Conversion research: first impressions, speed economics, landing-page anatomy, trust signals, UX laws, copywriting, forms, pricing psychology |
| `references/webflow-mastery.md` | Client-First class system, native-first building, interactions/GSAP craft, Webflow SEO/performance, and a tested Webflow Data API/MCP playbook with its gotchas |
| `references/anti-ai-look.md` | The visual tells that make a site read as AI-generated, and the escape moves (plus a self-test) |
| `references/qa-protocol.md` | The full pre-ship QA checklist and live-verification procedure |

## Install

Clone straight into your Claude Code skills folder:

```bash
# macOS / Linux
git clone https://github.com/Monthan-Zero/claude-skill-web-mastery.git ~/.claude/skills/web-mastery

# Windows (PowerShell)
git clone https://github.com/Monthan-Zero/claude-skill-web-mastery.git "$env:USERPROFILE\.claude\skills\web-mastery"
```

That's it. The skill registers automatically in every Claude Code session.

## Use

- Type `/web-mastery` in any Claude Code chat to invoke it directly, or
- Just ask for website work — the skill triggers automatically on landing pages, Webflow builds, redesigns, audits, and "make it look professional" requests.

Pairs well with an aesthetic-direction skill (it deliberately handles the *success and execution* side rather than picking visual styles).
